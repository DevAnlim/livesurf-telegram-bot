# LiveSurf Telegram Bot

Демо-бот для управления LiveSurf API через Telegram.
Использует `livesurf-sdk` и хранит пользовательские API-ключи в `bot_config.json`.

Это демонстрационный бот работающий на LiveSurf SDK (Python)

Установка:
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Запуск:
python3 bot.py
